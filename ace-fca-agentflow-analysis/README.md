## TL;DR

- **ACE-FCA** (HumanLayer): Separate phases (Research → Plan → Implement) to prevent context explosion in large codebases
- **AgentFlow** (Stanford): Separate modules (Planner → Executor → Verifier → Generator) to improve task execution quality
- **Key insight**: Create bottlenecks in information flow to discard noise and transmit only signal
